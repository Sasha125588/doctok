# 🚀 DocTok

**DocTok** — це навчальний додаток у форматі короткого контенту, натхненний TikTok, але замість відео користувач переглядає технічні статті з MDN.

Ідея полягає в тому, щоб зробити навчання швидким, зручним і захоплюючим через swipe-навігацію та мікроконтент.

---

## ✨ Основна ідея

DocTok перетворює документацію у формат стрічки:

- 📱 1 пост = 1 ідея
- ⚡ швидке споживання контенту
- 🔁 нескінченна стрічка
- 🧠 навчання без перевантаження

---

## 🧭 Навігація

DocTok використовує інтуїтивну swipe-навігацію:

- ↕ **Вертикальний свайп**
  - наступний пост
  - випадкова тема (як у TikTok)

- ↔ **Горизонтальний свайп**
  - наступний пост у межах тієї ж теми
  - послідовне вивчення

---

## 🧩 Типи контенту

Кожен пост має тип (`kind`), що визначає його роль:

- `summary` — коротке пояснення теми
- `example` — приклади коду
- `fact` — цікаві факти або деталі
